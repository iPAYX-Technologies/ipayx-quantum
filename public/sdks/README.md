# iPayX Client SDKs

Official client libraries and tools for integrating iPayX Protocol.

## Available SDKs

### TypeScript SDK
```bash
npm install @ipayx/sdk
```
- Full TypeScript/JavaScript support
- Promise-based async API
- Type definitions included
- [Documentation](typescript/README.md)

### Python SDK
```bash
pip install ipayx
```
- Pythonic interface
- Type hints support
- Async/await compatible
- [Documentation](python/README.md)

### CLI
```bash
npm install -g ipayx-cli
# or
npx ipayx-cli help
```
- Command-line interface
- Quick testing and debugging
- Scriptable automation
- [Documentation](cli/README.md)

### Embeddable Widget
```html
<iframe src="https://ipayx.ai/sdks/widget/ipayx_widget.html"></iframe>
```
- No dependencies
- Minimal styling
- Easy customization
- [View Widget](widget/ipayx_widget.html)

### Smart Contracts
```solidity
import "./IpayxRegistry.sol";
```
- On-chain payment references
- EVM-compatible
- Gas-optimized
- [View Contract](../contracts/IpayxRegistry.sol)

## Quick Start

1. **Get an API key** at [ipayx.ai/signup](https://ipayx.ai/signup)
2. **Choose your SDK** from the options above
3. **Install and initialize**
4. **Start building**
